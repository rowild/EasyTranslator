Key Features
------------
- Fast local neural text-to-speech
- 40+ languages supported
- Multiple interfaces (CLI, HTTP, Python, C++)
- GPU acceleration support (CUDA)
- Multi-speaker voices available
- ONNX Runtime for cross-platform compatibility
- Embedded espeak-ng for phonemization

Technology Stack
----------------
- Core: C++ (92.8%)
- Python Bindings: Python (6.6%)
- Model Format: ONNX Runtime
- Phonemization: espeak-ng (embedded)
- Voice Training: VITS (Variational Inference TTS)

Installation
------------
Quick Install:
  pip install piper-tts

With HTTP Support:
  pip install piper-tts[http]

With GPU Support:
  pip install piper-tts onnxruntime-gpu

System Requirements
-------------------
Minimum:
  - Python 3.7+
  - 512 MB RAM
  - 50 MB disk space per voice model

Recommended:
  - Python 3.11+
  - 2 GB RAM
  - 500 MB disk space (for multiple voices)

Optional:
  - CUDA-capable GPU for acceleration
  - 8 GB GPU RAM for high-quality synthesis

Quick Start Examples
--------------------

CLI:
  python3 -m piper -m en_US-lessac-medium -f output.wav -- 'Hello, world!'

HTTP Server:
  python3 -m piper.http_server --model en_US-lessac-medium
  curl -X POST -H 'Content-Type: text/plain' --data 'Test' -o test.wav http://localhost:5000/

Python API:
  from piper import PiperVoice
  voice = PiperVoice.load('en_US-lessac-medium.onnx')
  voice.synthesize('Hello from Python!', 'output.wav')

Docker:
  docker run -d -p 5000:5000 piper-tts-server
